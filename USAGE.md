# Usage

1. Open `index.html` in your browser.
2. Pick a day in the calendar to see its tasks.
3. Tick the checkboxes (images, videos, publish, etc.). Every tick saves instantly in your browser and shifts the day color from red (none) → amber (partial) → green (complete).
4. Add new task types in the "Task presets" area; they appear for every day automatically.
5. Use Export/Import for backups or moving data between browsers.
6. Use Clear All to reset local data in one click if you want a fresh start.
