# Usage

1. Open `index.html`.
2. Click a day to see its checklist.
3. Tick completed items. Saving happens immediately in your browser.
4. Use `Today` if you want to jump straight back to the current date.
5. Add new presets in the input below the selected-day panel.
6. Watch the insight cards at the top for streaks, active days, and note coverage.
7. Use `Reset day` to clear only the current date.
8. Use `Export JSON` before switching browsers or machines.
9. Use `Import JSON` to restore a backup.
10. Use `Clear all` only when you want a full reset of presets and saved progress.
