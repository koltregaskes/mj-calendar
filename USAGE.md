# Usage

1. Open `index.html`.
2. Click a day to see its checklist.
3. Tick completed items. Saving happens immediately in your browser.
4. Add new presets in the input below the selected-day panel.
5. Use `Reset day` to clear only the current date.
6. Use `Export JSON` before switching browsers or machines.
7. Use `Import JSON` to restore a backup.
8. Use `Clear all` only when you want a full reset of presets and saved progress.
