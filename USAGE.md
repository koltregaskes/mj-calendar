# Usage

1. Open `index.html` in your browser (or run `python -m http.server 8000` and visit the page).
2. Pick a day in the calendar to see its tasks.
3. Navigate with keyboard arrows inside the calendar grid, press Enter/Space to open a day, or use the skip link at the top to jump to the content quickly.
4. Tick the checkboxes (images, videos, publish, etc.). Every tick saves instantly in your browser and shifts the day color from red (none) → amber (partial) → green (complete). Mini dots on the calendar tile show which tasks are done, and the tile always reloads safely.
5. Add new task types with the input under the list; they appear for every day automatically.
6. Use Export/Import for backups or moving data between browsers.
7. Use Clear All to reset local data in one click if you want a fresh start.
